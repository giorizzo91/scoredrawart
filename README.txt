
SCOREDRAWART WEBSITE

Files included:
- index.html
- styles.css
- script.js
- products.json

How to use:
1. Upload all files to Netlify, Vercel or GitHub Pages.
2. Replace the product images and Etsy links in products.json.
3. Add Stripe or Shopify if you want direct checkout on your own site.
4. Edit branding/text inside index.html.

This version already works as a live responsive storefront.
